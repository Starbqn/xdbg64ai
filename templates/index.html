About Memory Editor

This is a barebones memory editor that allows you to attach to simulated processes, read and write memory values, and display them in a simple interface.

Features:

  • Process attachment capability
  • Memory reading and writing
  • Support for different data types (int, float, string)
  • Simulated process creation
